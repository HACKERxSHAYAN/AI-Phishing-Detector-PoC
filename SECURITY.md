## 🚨 Responsible Disclosure Policy

We take security seriously. If you discover any vulnerabilities, bugs, or potential exploits in **HACKERxSHAYAN AI-Phishing-Guard**, we encourage responsible disclosure.

### How to Report

1. **DO NOT** create public issues or pull requests for security vulnerabilities
2. **Email**: Contact the developer directly through appropriate channels
3. **Include**:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact assessment
   - Suggested remediation (if any)

### Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial Assessment**: Within 7 days
- **Fix Timeline**: Based on severity (critical issues prioritized)
